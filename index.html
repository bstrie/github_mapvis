This chart depicts the impact and geographic location of each contributor to the Rust compiler.

Location data is taken from each user's Github profile for users who have elected to disclose that information. Precise coordinates are found by running location strings through the MapQuest Nominatim API; for example, with a location of "Canada" you will be depicted as inhabiting a shack in remote Saskatchewan. Users without location data can be found floating near the Antarctic.

Impact is measured as the sum of lines added and lines deleted for all commits to the master branch. Github only makes this data available for the top 100 contributors to any given repository; for the remaining users, their location is denoted by a dot of the minimum size.
